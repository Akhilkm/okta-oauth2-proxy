# okta-oauth2-proxy
Add Auth to Any App with OAuth2 Proxy

### Create .env file in the below format
```
OIDC_ISSUER_URL=<<replace with your issuer>>
CLIENT_ID=<<client id>>
CLIENT_SECRET=<<client secret>
COOKIE_SECRET=<<cookie secret 32 byte>>
```
